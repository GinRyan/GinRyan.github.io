创建Bean后置处理器

使用本文上面的的例子，
我们有时候对于初始化的时候需要为当前配置文件中的所有的Bean再控制一些细节，比如计算等等、替换实例等等。那么我们可以使用bean的后置处理器在做这个工作。
而后置处理器在IoC容器中也是一个bean，但是这个Bean稍有特殊，它需要实现一个接口BeanPostProcessor，并且需要我们实现它的两个“拦截”方法： 

public Object postProcessBeforeInitialization(Object bean, String beanName)

public Object postProcessAfterInitialization(Object bean, String beanName)

这两个方法分别会在init-method的之前以及之后调用,参数Object bean是容器传递的实例， String beanName是容器传递的bean的id。

它不需要配置bean的id和属性，只需要配置类名即可。
